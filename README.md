# Team Chat / Slack Clone

A real-time collaboration platform with workspaces, channels, direct messaging, presence tracking, unread counts, and notification support.

# Features
- Workspace-based chat organization
- Channel messaging
- Direct messaging between users
- Online/offline presence tracking
- Unread message counts and notifications
- Authentication and protected routes
- Responsive chat layout with reusable UI components

# Tech Stack
- React
- TypeScript
- Tailwind CSS
- Axios
- Backend API integration

# Setup
1. Clone repo
2. Install dependencies
3. Add `.env`
4. Run frontend / backend
